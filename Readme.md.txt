# 📊 Excel Data Chatbot

An interactive web-based Excel data chatbot built using **Python** and **Streamlit**. Upload Excel files, filter data, generate summaries, group by columns, and ask basic analytical questions—all **without using external APIs**.

## 🚀 Features

- 📁 Upload Excel (.xlsx) files
- 🔎 View full dataset in the browser
- 🧮 Generate summary statistics (count, sum, mean)
- 🗂️ Filter data by specific columns
- 📊 Group data and apply aggregation
- ❓ Ask basic questions like:
  - "Total of Sales"
  - "Average of Revenue"
  - "Maximum of Quantity"
  - "Show data where Category is Electronics"
  - "How many rows?"

## 🛠️ Built With
- [Python](https://www.python.org/)
- [Streamlit](https://streamlit.io/)
- [Pandas](https://pandas.pydata.org/)
